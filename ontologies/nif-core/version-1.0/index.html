NLP Interchange Format (NIF) 2.0 release candidate

This document contains pointers to all the important resources relevant for the NIF Version 2.0. Namespace for the NIF Core Ontology is:
http://persistence.uni-leipzig.org/nlp2rdf/ontologies/nif-core# with the default prefix nif:
The computer center of the 600 year old University of Leipzig kindly provided us with a small amount of persistent web space for the publication of the NIF standard.

Status of this document

We are preparing a version 2.0 release candidate at the moment. Documentation and the ontologies are still subject to change at any time. Changes are tracked at our GitHub repository

Contents

For all ontologies, turtle is the default format with additional comments in the code. All other formats are generated.

  • Inference Model in Turtle, OWL, HTML (coming soon)
